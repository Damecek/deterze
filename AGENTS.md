# AGENTS.md

## Scope
Tyto instrukce platí pro celý repozitář.

## Cíl projektu
Tento repozitář je statický one-page web kapely Deterze.
Priorita je jednoduchost, čitelnost a snadná editace i bez build nástrojů.

## Technické zásady
- Používej pouze plain HTML, CSS a JavaScript.
- Nevkládej frameworky ani build toolchain, pokud to není výslovně požadováno.
- Udržuj strukturu projektu minimální (`index.html`, `styles.css`, `script.js`, `public/`).
- Preferuj obsah v češtině.

## Obsahové zásady
- Zachovej sekce: O kapele, Členové, Galerie, Kontakt.
- Koncerty odkazuj na Facebook/Instagram (nezakládej samostatnou sekci s termíny).
- Nové texty připravuj jako reálný návrh + zřetelné placeholdery pro pozdější doplnění.

## Deployment
- Předpokládaný deployment je GitHub Pages s custom doménou (`CNAME`).
- Pokud měníš deployment postup, aktualizuj i `README.md`.
